Installing on Ubuntu

LocalWiki is available as a prebuilt package for Ubuntu 10.04 LTS or newer.

Just open a terminal and run these commands:

  • $
  • $
  • $
  •  
  • $
  • sudo apt-get install python-software-properties
  • sudo apt-add-repository ppa:localwiki
  • sudo apt-get update
  •  
  • sudo apt-get install localwiki

That's it! Follow the prompts to configure LocalWiki.

After installing, be sure to add your new project to the LocalWiki Guide.

Upgrading

When a new version of LocalWiki is released, you should upgrade to it. If you installed using the Ubuntu instructions above, then upgrading to a new release is very simple.

Run these commands to upgrade:

  • $
  • $
  • sudo apt-get update
  • sudo apt-get upgrade

This will upgrade all of your packages, including LocalWiki. All of your data will be preserved during the upgrade.

Installing on other systems

If you have the option, we strongly recommend installing LocalWiki on Ubuntu 10.04 LTS. It makes both installing and keeping LocalWiki up to date much easier.

If you still want to install on a different operating system, please read our documentation for detailed instructions.
